## Pipeline Rubric
    - Weight = 25%
    - Meets Expectations (Proficient): Successfully ingests and cleans a large, structured dataset. Implements a functional ETL process.
    - Exemplary: Demonstrates custom schema mapping
## Technical Audit Rubric
    - Weight = 20%
    - Meets Expectations (Proficient): Code is functional and well-documented. 
    - Exemplary: Documentation includes corrections identifying specific instances where code was improved over time - in commented out code where improvements were made.
## Scalability & Stress Testing Rubric
    - Weight = 20%
    - Meets Expectations (Proficient): Code runs successfully on the provided dataset without crashing. Basic understanding of distributed computing.
    - Exemplary: Demonstrates optimizations used for scalability.
## Analytical Depth & Insight Rubric
    - Weight = 20%
    - Meets Expectations (Proficient): Accurately identifies trends and correlations.
    - Exemplary: Moves beyond surface-level stats to find and explain a non-obvious anomoly. Connects findings to complex real-world domain context or economic/social theory, regardless if it confirms or differs.
## Ethical Guardrails & Governance Rubric
    - Weight = 15%
    - Meets Expectations (Proficient): Basic documentation of data processing.
    - Exemplary: Tests the analytics for algorithmic bias, Implements, defends or proposes alternative approaches to data processing.
